# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-03-30

### Added
- Interactive map view for `.gpx` files powered by Leaflet and OpenStreetMap
- Route drawn in Strava orange with start (green) and finish (red) markers
- Stats panel with distance, elevation gain, duration and pace
- Automatic track name and date from GPX metadata
- GitHub Actions CI pipeline (type check + build on every push/PR)
- GitHub Actions release pipeline (creates a release with artifacts on version tag)
